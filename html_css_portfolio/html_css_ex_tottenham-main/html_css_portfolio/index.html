토트넘 홋스퍼 2024/25 시즌티켓 구매는 이렇게 하세요

원 핫스퍼 멤버십 회원은 다음과 같은 다양한 혜택을 누릴 수 있습니다.
  • 웰컴 기프트
  • 경기 티켓 구매권
  • 티켓 교환 - 공식 티켓 재판매 플랫폼 이용 가능
  • One Hotspur Moments - 리워드 및 체험 프로그램
  • 클럽 할인 및 혜택
  • 파트너사 혜택

5파운드를 추가로 지불하시면 원 핫스퍼+ 멤버십을 구매하실 수 있으며, 시즌 티켓
대기자 명단에 자동으로 추가되어 원 핫스퍼 멤버보다 24시간 먼저 티켓을 구매할
수 있는 우선권을 드립니다.
